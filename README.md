# ReliefDrop

**One-line description:** A transparent, instant disaster relief distribution system using Stellar and Soroban.

**Problem:** Disaster victims wait weeks for physical aid while NGOs struggle with slow, opaque cash logistics.
**Solution:** NGO deposits USDC into a smart contract, whitelists eligible citizens, who then instantly claim their exact monetary allocation straight to their wallets.

**Timeline:** 3-Day Hackathon Project
**Stellar Features Used:** Soroban smart contracts, USDC transfers, Trustlines.
**Vision and Purpose:** To eliminate corruption and supply-chain bottlenecks in disaster relief, empowering victims with immediate purchasing power.

### Prerequisites
* Rust toolchain (edition 2021)
* Soroban CLI (`soroban-cli v20.0.0` or higher)
